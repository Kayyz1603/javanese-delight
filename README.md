
About
========
Javanese Delight is an expansion to Farmer's delight, adding various crops, ingredients, and dishes from
the island of Java, Indonesia. Additionally, foods may give you useful special effects!


Installation
========
**Versions supported:** NeoForge 1.21.1  
**Dependencies:** Farmer's Delight 1.2.4+  
The mod .jar file can be downloaded through CurseForge and Modrinth.
